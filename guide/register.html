注册 / 登录 ​

按照本页完成 CCWSPro 账号创建和登录，并确认控制台、余额和 API Key 入口可以正常访问。目前仅支持账号密码注册登录方式。

注册入口 ​

  1. 打开 CCWSPro 官网。
  2. 点击右上角的「注册」或「开始使用」。
  3. 按页面提示填写账号和密码。
  4. 再次输入密码完成确认。
  5. 点击「创建账户」。

登录入口 ​

注册完成后，返回登录页输入用户名或邮箱和密码。

  1. 打开登录页。
  2. 输入用户名或邮箱。
  3. 输入密码。
  4. 点击「登录」进入控制台。

注册前准备 ​

项目说明
账号和密码用于注册和后续登录，请妥善保存
浏览器推荐使用 Chrome、Edge、Safari 的最新版本
网络环境如果页面长时间无响应，先确认网络和 DNS 是否正常

提示

如果页面提示需要邀请码或人工审核，请根据页面说明提交申请，或加入售后群确认开通方式。

注册后检查 ​

  • 能正常进入控制台。
  • 能看到账号余额或套餐状态。
  • 能进入 API Key 管理页面。
  • 能找到帮助入口或售后联系方式。

下一步 ​

注册并确认可以登录后，继续完成 充值 / 续费。
