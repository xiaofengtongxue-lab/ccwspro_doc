# 疑难解答总览

本目录收集 CCWS 用户在平台注册、账号充值、API Key、工具配置和工具调用中常见的问题。

## 排查顺序

1. 先确认账号状态和余额。
2. 再确认 API Key 是否有效。
3. 检查 Base URL 和模型名称。
4. 查看工具日志中的完整报错。
5. 如果仍无法解决，带上截图和日志联系支持。
