# 注册账号

按照本页完成 CCWSPro 账号创建，并确认控制台、余额和 API Key 入口可以正常访问。目前仅支持账号密码注册登录方式。

![注册登录页面](/images/screenshots/register-login.svg)

## 注册入口

1. 打开 CCWSPro 官网。
2. 点击右上角的「注册」或「开始使用」。
3. 按页面提示填写账号和密码。
4. 完成必要校验后登录控制台。

## 注册前准备

| 项目 | 说明 |
| --- | --- |
| 账号和密码 | 用于注册和后续登录，请妥善保存 |
| 浏览器 | 推荐使用 Chrome、Edge、Safari 的最新版本 |
| 网络环境 | 如果页面长时间无响应，先确认网络和 DNS 是否正常 |

::: tip 提示
如果页面提示需要邀请码或人工审核，请根据页面说明提交申请，或加入售后群确认开通方式。
:::

## 注册后检查

- 能正常进入控制台。
- 能看到账号余额或套餐状态。
- 能进入 API Key 管理页面。
- 能找到帮助入口或售后联系方式。
