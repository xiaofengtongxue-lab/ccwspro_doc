# 注册账号

本页用于说明用户如何在 CCWSPro 完成账号创建。正式发布前，请把截图替换为你自己站点的真实注册页面。

![注册登录页面](/images/screenshots/register-login.svg)

## 注册入口

1. 打开 CCWSPro 官网。
2. 点击右上角的「注册」或「开始使用」。
3. 选择手机号、邮箱或第三方账号登录方式。
4. 按页面提示完成验证码校验和基础信息填写。

## 注册前准备

| 项目 | 说明 |
| --- | --- |
| 邮箱或手机号 | 用于接收验证码和后续通知 |
| 浏览器 | 推荐使用 Chrome、Edge、Safari 的最新版本 |
| 网络环境 | 如果页面长时间无响应，先确认网络和 DNS 是否正常 |

::: tip 提示
如果你的站点只开放邀请注册，请在这里补充邀请码获取方式和审核规则。
:::

## 注册后检查

- 能正常进入控制台。
- 能看到账号余额或套餐状态。
- 能进入 API Key 管理页面。
- 能找到帮助入口或售后联系方式。
