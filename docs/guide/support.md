# 售后与支持

当用户在注册、充值、Key 配置或工具调用中遇到问题时，可以通过本页找到支持入口。

## 建议提供的信息

提交问题前，请准备：

- 账号邮箱或手机号。
- 问题发生时间。
- 报错截图或完整错误信息。
- 使用的工具名称和版本。
- 已尝试过的排查步骤。

## 支持渠道

| 渠道 | 用途 | 响应说明 |
| --- | --- | --- |
| 在线客服 | 账号、充值、基础配置问题 | 按实际服务时间填写 |
| 技术支持群 | 工具配置、接口调用问题 | 适合截图和日志排查 |
| 商务邮箱 | 企业采购、批量账号、私有化咨询 | 适合正式合作沟通 |

::: warning 发布前替换
请把这里的支持渠道替换成你自己的微信、企微、邮箱、飞书群或客服系统链接。
:::
