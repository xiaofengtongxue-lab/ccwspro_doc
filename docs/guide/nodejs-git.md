# Node.js 与 Git 环境

如果用户需要在本地使用 Codex 或其他命令行工具，建议先准备 Node.js 和 Git。

## Node.js

推荐安装 Node.js LTS 版本。

```bash
node -v
npm -v
```

如果能看到版本号，说明 Node.js 已安装成功。

## Git

```bash
git --version
```

如果能看到版本号，说明 Git 已安装成功。

## 常见建议

- macOS 用户可以使用 Homebrew 安装。
- Windows 用户可以使用官方安装包或 winget。
- 如果要管理多个 Node.js 版本，可以使用 nvm、fnm 或 Volta。
