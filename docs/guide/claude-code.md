# Claude Code 配置

本页用于说明如何把 CCWSPro 接入 Claude Code 或兼容的命令行工具。具体字段请以你实际支持的工具和模型为准。

## 准备信息

- CCWSPro 请求地址。
- CCWSPro API Key。
- 可用模型名称。
- 本地 Node.js / Git 环境。

## 配置思路

1. 打开 Claude Code 的配置文件或设置入口。
2. 填入 Base URL。
3. 填入 API Key。
4. 选择或填写默认模型。
5. 保存后重启终端。

## 验证方式

进入一个测试目录，发起简单问题：

```text
请读取当前目录结构，并说明这个项目是什么类型。
```

如果工具能正常响应，说明配置基本可用。

::: warning
不同版本的 Claude Code 配置方式可能变化。正式发布前请用你当前推荐的版本重新验证一遍。
:::
