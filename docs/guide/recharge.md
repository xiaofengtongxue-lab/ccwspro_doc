# 充值 / 续费

本页用于说明余额充值、套餐续费和支付结果确认流程。

![充值页面](/images/screenshots/recharge.svg)

## 充值流程

1. 登录 CCWSPro 控制台。
2. 进入「充值」「账单」或「套餐」页面。
3. 选择充值金额或套餐类型。
4. 选择支付方式并完成付款。
5. 返回控制台确认余额或套餐状态已更新。

## 付款后未到账

遇到付款后未到账，建议按下面顺序排查：

- 等待 1 到 3 分钟后刷新页面。
- 检查支付订单是否成功。
- 保存订单号、付款截图和账号信息。
- 联系 QQ 售后群或管理员处理。

::: tip 建议
如需开票、企业付款或批量充值，请通过售后群或商务合作入口联系工作人员。
:::
