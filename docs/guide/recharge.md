# 充值 / 续费

本页用于说明余额充值、套餐续费和支付结果确认流程。

![充值页面](/images/screenshots/wallet-recharge.png)

## 充值流程

1. 登录 CCWSPro 控制台。
2. 进入「钱包」页面。
3. 在「添加资金」区域选择固定金额，或填写自定义金额。
4. 选择支付方式，目前页面支持支付宝。
5. 确认待支付金额后完成付款。
6. 返回控制台确认余额已更新。

## 兑换码兑换

如果你有兑换码，可以在钱包页面底部完成兑换。

![兑换码兑换](/images/screenshots/wallet-redeem-code.png)

1. 找到「有兑换码吗？」区域。
2. 输入兑换码。
3. 点击「兑换额度」。
4. 兑换完成后刷新页面，确认余额或额度已更新。

## 付款后未到账

遇到付款后未到账，建议按下面顺序排查：

- 等待 1 到 3 分钟后刷新页面。
- 检查支付订单是否成功。
- 保存订单号、付款截图和账号信息。
- 联系 QQ 售后群或管理员处理。
