# 商务合作

如需企业采购、渠道合作、培训服务、工作流定制或软件定制，可以先整理需求，再联系工作人员。

## 适合咨询的场景

- 企业采购。
- 渠道合作。
- 培训服务。
- 工作流定制。
- 软件定制。

## 沟通前建议准备

| 信息 | 示例 |
| --- | --- |
| 团队规模 | 5 人、20 人、100 人 |
| 合作类型 | 企业采购、渠道合作、培训服务、工作流定制、软件定制 |
| 使用场景 | AI 编程、代码审查、知识库问答、Agent 应用 |
| 预算方式 | 充值余额、月度套餐、年度采购 |

## 联系方式

请加入 QQ 售后群，并说明「商务合作」或「企业采购」需求。工作人员会根据团队规模、合作类型和具体需求继续对接。
