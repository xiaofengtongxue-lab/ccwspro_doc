# 商务合作

本页用于承接企业采购、渠道合作、培训服务或私有化部署咨询。

## 适合咨询的场景

- 团队需要统一采购账号或套餐。
- 企业希望接入内部研发工作流。
- 需要 AI 编程工具培训、落地陪跑或课程合作。
- 需要私有化、专属网关、账单管理或权限隔离方案。

## 沟通前建议准备

| 信息 | 示例 |
| --- | --- |
| 团队规模 | 5 人、20 人、100 人 |
| 使用场景 | AI 编程、代码审查、知识库问答、Agent 应用 |
| 预算方式 | 充值余额、月度套餐、年度采购 |
| 合规要求 | 发票、合同、数据隔离、访问审计 |
