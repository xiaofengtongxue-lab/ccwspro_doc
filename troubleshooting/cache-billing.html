缓存与计费 ​

余额显示未更新 ​

处理顺序：

  1. 刷新控制台。
  2. 退出重新登录。
  3. 等待支付回调完成。
  4. 联系支持并提供订单号。

调用记录延迟 ​

调用记录可能存在短暂延迟。排查问题时，优先以接口返回结果和工具日志为准。

费用异常 ​

请记录：

  • 账号信息。
  • 调用时间。
  • 使用模型。
  • 预期费用和实际费用。
  • 相关请求 ID 或日志截图。
